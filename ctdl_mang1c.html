Cấu trúc dữ liệu mảng 1 chiều c++ - list Python

1. Khái niệm

- Mảng một chiều là cấu trúc dữ liệu tuyến tính, là cấu trúc dữ liệu bao gồm một dãy các phần tử có cùng kiểu.
- Trong python, dữ liệu mảng một chiều có thể được biểu diễn bằng danh sách (list) với các phần tử cùng kiểu dữ liệu.

  • Truy cập bằng chỉ số
  • Các phần tử liên tiếp trong bộ nhớ

Tính chất:

  • Thuận tiện trong việc truy xuất dữ liệu
  • Các phần tử trong mảng được gán chỉ số
  • Kích thước cố định, không thể mở rộng và thu hẹp

2. Cú pháp khai báo mảng:

2.1 Cú pháp khai báo mảng Python

tên_biến=[phần_tử_1, phần_tử_2, phần_tử_3, ...]

Ví du:

  • Mảng rỗng
    • my_list=[]
  • Các phần tử có giá trị ngẫu nhiên
    • a=[0]*100
  • Mảng có giá trị khởi tạo
    • numbers=[1,2,3,4,5]
    fruits=["táo","chuối","cam"]

2.2 Cú pháp khai báo mảng trong c++

Kiểu_Dữ_liệu tên_mảng[Số_lượng_phần_tử]

Ví dụ:

  • Mảng với các giá trị ngẫu nhiên
    • int a[100];
  • Các phần tử có giá trị cụ thể
    • b[6]={2,8,0,4,2,5};
  • Mảng kí tự
    • char arr[100]={'2','8','t','e'};

3. Các thao tác trên mảng:

3.1 Các thao tác trên danh sách python

✼ Duyệt danh sách:

  • Các phần tử trong danh sách (list) được đánh số từ 0 đến N-1
  • Truy cập tới từng phần tử bằng: ten_danh_sach[chi_so]
  • Có thể: truy xuất, thay đổi giá trị, duyệt qua từng phần tử bằng vòng lặp
    •       # Khởi tạo danh sách
      n = 10
      a = [1, 5, 2, 3, 4, 8, 7, 6, 9, 10]

      print("Duyet thuan:", end=" ")
      for i in range(n):
         print(a[i], end=" ")

      print("\nDuyet nguoc:", end=" ")
      for i in range(n - 1, -1, -1):
         print(a[i], end=" ")

      # Danh sách ký tự
      b = ['t','u','d','i','e','n','l','a','p','t','r','i','n','h']

      print("\nDanh sách ki tu:", end=" ")
      for i in range(len(b)):
         print(b[i], end=" ")

      Output : 
      Duyet thuan : 1 5 2 3 4 8 7 6 9 10 
      Duyet nguoc : 10 9 6 7 8 4 3 2 5 1
      Danh sách ki tu : t u d i e n l a p t r i n h

✼ Thay đổi giá trị của các phần tử trong danh sách:

      # Khởi tạo danh sách
      b = [1, 5, 10, 4, 2, 6]

      # Thay đổi giá trị
      b[0] = 30
      b[1] = 50
      b[2] = 24
      b[3] += 10
      b[4] *= 3
      b[5] //= 4   # dùng // để lấy phần nguyên giống C++

      # In kết quả
      print("Mang sau khi thay doi:", end=" ")
      for i in range(len(b)):
         print(b[i], end=" ")
      
      Output:
      Mang sau khi thay doi: 30 50 24 14 6 1

✼ Nhập danh sách từ bàn phím:

    Nhập danh sách từ bàn phím với kích thước cố định

          # Khai báo mảng tối đa 1000 phần tử
      a = [0] * 1000
      n = int(input("Nhap n: "))
      for i in range(n):
         a[i] = int(input(f"a[{i}] = "))
      print("Mang vua nhap:", end=" ")
      for i in range(n):
         print(a[i], end=" ")

    Nhập danh sách từ bàn phím với kích thước vừa đủ

          n = int(input("Nhap n: "))
      a = []  # mảng rỗng
      for i in range(n):
         x = int(input(f"a[{i}] = "))
         a.append(x)
      print("Mang vua nhap:", end=" ")
      for x in a:
         print(x, end=" ")

    Chú ý :Python linh hoạt hơn C++, không cần khai báo kích thước, dùng append() để thêm phần tử.

3.2 Các thao tác trên mảng c++

✼ Duyệt mảng:

  • Các phần tử được đánh số tính từ 0, ví dụ mảng có N phần tử chỉ số sẽ được đánh số từ 0 đến N-1.
  • Để truy cập tới từng phần tử của mảng ta dùng cú pháp: tên_mảng[chỉ_số]
  • Thông qua chỉ số ta có thể truy xuất, thay đổi... giá trị của từng phần tử trong mảng tại chỉ số tương ứng.
  • Để duyệt qua các phần tử trong mảng thì bạn duyệt qua từng chỉ số:
    •       #include 
      using namespace std;

      int main(){
         int n = 10;
         int a[10] = {1,5,2,3,4,8,7,6,9,10};
         cout << "Duyet thuan : ";
         for(int i = 0; i < n; i++){
            cout << a[i] << " ";
         }
         cout << "\nDuyet nguoc : ";
         for(int i = n - 1; i >= 0; i--){
            cout << a[i] << " ";
         }
         char b[14] = {'t','u','d','i','e','n','l','a','p','t','r','i','n','h'};
         cout << "\nMang ki tu : ";
         for(int i = 0; i < sizeof(b); i++){
            cout << b[i] << ' ';
         }
         return 0;
      }

      Output : 
      Duyet thuan : 1 5 2 3 4 8 7 6 9 10 
      Duyet nguoc : 10 9 6 7 8 4 3 2 5 1
      Mang ki tu : t u d i e n l a p t r i n h

✼ Thay đổi giá trị trong mảng:

      #include 
      using namespace std;

      int main(){
         int b[6] = {1, 5, 10, 4, 2, 6};
         b[0] = 30;
         b[1] = 50;
         b[2] = 24;
         b[3] += 10;
         b[4] *= 3;
         b[5] /= 4;
         cout << "Mang sau khi thay doi : ";
         for(int i = 0; i < 6; i++){
            cout << b[i] << ' ';
         }
         return 0;
      }

      Output : 
      Mang sau khi thay doi : 30 50 24 14 6 1

✼ Nhập mảng từ bàn phím:

    Nhập mảng từ bàn phím với kích thước cố định

  • Để nhập mảng từ bàn phím, đầu tiên nhập số lượng phần tử trong mảng sau đó nhập từng phần tử trong mảng từ đầu tới cuối.
  • Khi nhập mảng từ bàn phím bạn có thể khai báo mảng có kích thước cố định (1000, 2000, 100...) hoặc khai báo dựa trên số lượng phần tử mà đề bài cho.
    •       #include 
      using namespace std;

      int main(){
         //Lưu ý là mảng a chỉ lưu được tối đa 1000 phần tử
         int n, a[1000];
         cout << "Nhap n : ";
         cin >> n;
         for(int i = 0; i < n; i++){
            cout << "a[" << i << "] = ";
            cin >> a[i]; 
         }
         cout << "Mang vua nhap : ";
         for(int i = 0; i < n; i++){
            cout << a[i] << ' ';
         }
         return 0;
      }

    Chú ý : Khi bạn sử dụng mảng cố định thì cần lưu ý tới dữ liệu của đề bài, ví dụ trong code trên khai báo mảng a[1000] thì nó sẽ đúng nếu ta sử dụng mảng a với các bài toán có số lượng phần tử ≤ 1000. Trường hợp đề bài cho nhiều hơn 1000 phần tử thì sẽ bị sai vì không đủ bộ nhớ.

    Nhập mảng từ bàn phím với kích thước vừa đủ

          #include 
      using namespace std;

      int main(){
         //Lưu ý là mảng a chỉ lưu được tối đa 1000 phần tử
         int n;
         cout << "Nhap n : ";
         cin >> n;
         int a[n];
         for(int i = 0; i < n; i++){
            cout << "a[" << i << "] = ";
            cin >> a[i]; 
         }
         cout << "Mang vua nhap : ";
         for(int i = 0; i < n; i++){
            cout << a[i] << ' ';
         }
         return 0;
      }
